💄 Descripción del proyecto
Este proyecto corresponde al sitio web empresarial VNABelleza, desarrollado con el CMS Joomla 5.3.4.  
El objetivo es demostrar el uso de herramientas low-code y zero-code en el desarrollo web, implementando un portal para una empresa de productos de skincare (cuidado facial y corporal) en Costa Rica.



🧩 Estructura del proyecto
- `administrator/` → panel administrativo del CMS  
- `templates/` → plantillas visuales del sitio  
- `images/` → recursos gráficos y logotipos  
- `configuration.php` → archivo principal de configuración (no incluido en Git)  
- `installation/` → carpeta de instalación (debe eliminarse tras completar la instalación)  



⚙️ Instalación local (pasos básicos)
1. Instalar XAMPP y activar Apache y MySQL.  
2. Crear la base de datos `vnabelleza_db` en phpMyAdmin.  
3. Clonar este repositorio en `C:\xampp\htdocs\vnabelleza`.  
4. Abrir en el navegador: [http://localhost:8080/vnabelleza](http://localhost:8080/vnabelleza)  
5. Completar el asistente de instalación de Joomla.  
6. Eliminar la carpeta `/installation` tras finalizar.
